iframe内容示例

这是一个嵌入在弹窗中的iframe内容。

你可以在这里放置任何HTML内容，包括表单、图表、表格等。

设置操作按钮区示例：

在Iframe弹窗网页内设置.layer-bottom-btns 可让其内部的按钮展示在弹窗底部的操作按钮区。例如：

<div class="layer-bottom-btns">
  <button class="btn" onclick="parent.postMessage({type: 'close'}, '*')">关闭弹窗</button>
  <button class="btn" onclick="sendCallback()">发送回调</button>
</div>



弹窗父子页面通讯示例：

在Iframe弹窗网页内调用父页面的方法 。例如：

// Iframe调用父页面的test方法
parent.test();

// 父页面调用Iframe的test方法
let testModal = Modal.iframe('Iframe弹窗', '/demo/modal/iframe-content', 600, 400);
// iframe 完全加载后回调调用
testModal.onIframeComplete((_modal) => {
    console.log('complete',_modal);
    // 父页面调用Iframe网页中的test方法并传参
    _modal.callIframeFunction('test', '参数1','参数2',...)
})
